# WEST Tools

WEST ships with a suite of companion applications for editing models, managing data, and designing plants.

| Tool | Purpose |
|---|---|
| [Model Editor](model-editor.md) | Write and edit process models in MSL; Gujer matrix editor; automatic code generation |
| [Block Editor](block-editor.md) | Create and manage block icon libraries used in layouts |
| [Data Editor](data-editor.md) | Manage input data files (time-series, influent files) |
| [Designer](designer.md) | Size a treatment plant using conventional design rules (ATV, Metcalf & Eddy) |
| [Unit Editor](unit-editor.md) | Manage the WEST unit table and conversion factors |

**Source:** WEST User Guide — Tools User Guide section (Chapters 7–11).
