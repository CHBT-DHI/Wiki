# Experiment Types

WEST supports six experiment types. All are built on top of a base Dynamic simulation and accessed via **Project | Virtual Experiments**.

| Experiment | Purpose | Source |
|---|---|---|
| Steady-state & Dynamic | Base simulation modes | [Running Simulations](../how-to/running-simulations.md) |
| [Sensitivity Analysis](sensitivity-analysis.md) | Quantify how outputs respond to parameter changes | Tutorial ch 9–10 |
| [Parameter Estimation](parameter-estimation.md) | Automatic calibration against measured data | Tutorial ch 11 |
| [Scenario Analysis](scenario-analysis.md) | Compare multiple parameter sets in one run | Tutorial ch 12 |
| [Uncertainty Analysis](uncertainty-analysis.md) | Propagate parameter uncertainty to outputs | Tutorial ch 13 |

!!! info "WEST+ required"
    Local Sensitivity Analysis, Global Sensitivity Analysis, Parameter Estimation, Scenario Analysis, and Uncertainty Analysis require the **WEST+** licence.
