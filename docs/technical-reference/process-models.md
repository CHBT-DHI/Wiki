**Summary:** The biological process models available in WEST, when to use each, and their key assumptions.

|---|---|---|
| ASM1 | Carbon and nitrification/denitrification | N only |
| ASM2d | Biological phosphorus removal | N and P |
| ASM3 | Internal storage, improved N model | N only |
