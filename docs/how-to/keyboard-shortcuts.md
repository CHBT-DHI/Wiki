# Keyboard Shortcuts

**Summary**: Quick reference for WEST keyboard shortcuts.

**Source**: WEST User Guide §2.11.

---

### General

| Shortcut | Action |
|---|---|
| `Ctrl+N` | New project |
| `Ctrl+O` | Open project |
| `Ctrl+S` | Save project |
| `Ctrl+Z` | Undo |
| `Ctrl+Y` | Redo |
| `Ctrl+C` | Copy selected block(s) |
| `Ctrl+V` | Paste |
| `Ctrl+X` | Cut |
| `Delete` | Delete selected block or connection |
| `Ctrl+A` | Select all blocks on current sheet |
| `F1` | Open help |
| `Esc` | Cancel current operation / deselect |

---

### Layout canvas

| Shortcut | Action |
|---|---|
| `Ctrl+scroll` | Zoom in/out |
| `Middle mouse drag` | Pan canvas |
| `Space+drag` | Pan canvas (alternative) |
| `Ctrl+Shift+F` | Fit layout to window |
| `Ctrl+G` | Group selected blocks |
| `Ctrl+Shift+G` | Ungroup |
| `Ctrl+L` | Open Layers window |
| `F2` | Rename selected block |
| `Ctrl+D` | Duplicate selected block |
| `Arrow keys` | Nudge selected block(s) by 1 grid unit |
| `Shift+Arrow` | Nudge by 10 grid units |

---

### Simulation

| Shortcut | Action |
|---|---|
| `F5` | Run steady-state simulation |
| `F6` | Run dynamic simulation |
| `F7` | Stop simulation |
| `F8` | Initialise / warm-start from current state |
| `Ctrl+R` | Open Run Settings dialog |

---

### Results and plots

| Shortcut | Action |
|---|---|
| `Ctrl+E` | Export selected plot or table |
| `Ctrl+P` | Print current sheet |
| `Double-click plot` | Open plot properties |
| `Right-click plot` | Context menu (add series, export, copy) |

---

### Model Explorer

| Shortcut | Action |
|---|---|
| `Ctrl+F` | Search in Model Explorer |
| `Drag to sheet` | Add variable as a plot on the current sheet |
