# Running Simulations

**Summary:** How to configure and run steady-state and dynamic simulations in WEST.

**Prerequisites:**

- A project with at least one layout containing connected process units.  
  See [Quick Start Tutorial](../starter-kit/quick-start.md) if you have not built a model yet.

---

## Simulation types

| Type | Use when… |
|---|---|
| **Steady-state** | You want the equilibrium operating point of the plant |
| **Dynamic** | You want to see how the plant responds to varying influent or control actions over time |
