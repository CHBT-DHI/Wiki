# Quick Start Tutorial

**Summary:** Build and run a simple activated sludge model in WEST from scratch. Estimated time: 20–30 minutes.

**Prerequisites:**

- WEST is installed and licensed. See [Installation](../manuals/installation.md) if not.
- No prior WEST experience required.

---

## What you will build

A single-reactor activated sludge model with a simple settler, driven by a constant influent. By the end you will have run a steady-state simulation and viewed the effluent quality results.

---

## Step 1 — Create a new project

1. Open WEST.
2. Select **File → New Project**.
3. Give the project a name (e.g. `QuickStart`) and choose a save location.
4. Click **OK**.
